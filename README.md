# Vanilla JS + TS + Tailwindcss + json-server TODO List Project

## 활용 기술

- Vanilla JS
- TypeScript
- Tailwindcss
- Json Server
- Vite

## 설명

- Vanilla JS 로 구현한 SPA 느낌의 TODO List

## 기능 설명

- TODO List 추가
- TODO List 출력
- TODO List 수정
- TODO List 삭제
- 각 상태에 따른 실시간 변경 -> 페이지 새로고침 없이 가능

## 실행 방법

```bash

git clone

npm install

// server on

npm run server

// open browser

npm run dev

```
